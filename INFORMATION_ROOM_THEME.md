# 📰 NR2 정보방 테마 변경 완료 ✅

## 변경 사항

### 1. 브랜드 리뉴얼
- **이전**: NR2 커뮤니티
- **변경**: 📰 NR2 정보방
- **컨셉**: 뉴스와 정보를 공유하는 실시간 정보 커뮤니티

### 2. 게시판 이름 변경
| 이전 | 변경 | 설명 |
|------|------|------|
| 자유게시판 | **자유정보** | 다양한 정보를 자유롭게 공유 |
| LEFT게시판 | **LEFT정보** | 진보 성향 뉴스와 정보 |
| RIGHT게시판 | **RIGHT정보** | 보수 성향 뉴스와 정보 |
| 가짜뉴스게시판 | **팩트체크** | 가짜뉴스 검증과 사실 확인 |

### 3. 새로운 기능: 📰 속보
실시간 뉴스 속보 시스템 추가
- **URL**: `http://localhost:5001/news/breaking`
- **특징**:
  - 🔴 긴급속보 (우선순위 5+)
  - ⚠️ 중요속보 (우선순위 3-4)
  - 📰 일반속보 (우선순위 0-2)
  - 출처 표시
  - 우선순위별 자동 정렬

### 4. UI 개선
- **메인 페이지**: 그라데이션 카드 디자인
- **네비게이션**: 속보 메뉴 강조 (빨간색)
- **푸터**: "NR2 정보방" 브랜딩

## 새로운 데이터 구조

### BreakingNews 모델
```python
class BreakingNews(db.Model):
    id              # 속보 ID
    title           # 제목
    content         # 내용
    source          # 출처 (예: 연합뉴스, KBS, 자체취재)
    priority        # 우선순위 (0-10)
    is_active       # 활성화 여부
    created_at      # 작성일시
    updated_at      # 수정일시
    user_id         # 작성자 (관리자)
```

## 속보 관리 (관리자 전용)

### 속보 등록
1. 속보 페이지 접속
2. "속보 등록" 버튼 클릭
3. 제목, 내용, 출처, 우선순위 입력
4. 등록

### 우선순위 설정
- **10**: 최우선 (최상단 고정)
- **5**: 긴급 (🔴 긴급속보 배지)
- **3**: 중요 (⚠️ 중요속보 배지)
- **0**: 일반 (📰 속보 배지)

### 속보 수정/삭제
- 각 속보 상세 페이지에서 수정/삭제 가능
- 활성화/비활성화 토글
- 비활성화 시 일반 사용자에게 보이지 않음

## 테스트 데이터

4. [중요] 새로운 기술 정책 발표, IT 업계 변화 예고

## 사용 시나리오

### 일반 사용자
1. 메인 페이지에서 "📰 속보" 클릭
2. 실시간 속보 목록 확인
3. 관심 있는 속보 클릭하여 상세 내용 확인
4. 각 게시판(자유정보, LEFT정보 등)에서 정보 공유

### 관리자
1. 속보 페이지에서 "속보 등록" 클릭
2. 제목, 내용, 출처, 우선순위 입력
3. 긴급한 뉴스는 우선순위 5 이상으로 설정
4. 등록 후 속보 목록 상단에 표시
5. 필요시 수정/삭제 또는 비활성화
